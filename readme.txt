Scans your pages for CamelCaseWords, or WikiWords, and links them to appropriate internal pages.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wikiwords.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== How It Works ==

1. Publish a page that contains words in camel case (or WikiWords) like IfYouLikePinaColadas or GettingCaughtInTheRain
2. If pages with such titles exist, the above WikiWords will be linked to them.
3. If pages with such titles don't exist, the above WikiWords will be linked to the post editor, prompting the user to create such a page.
4. That's it.
